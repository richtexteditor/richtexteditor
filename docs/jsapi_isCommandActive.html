editor.isCommandActive(CommandName)

Gets whether the specified command is active.

Syntax

editor.isCommandActive(CommandName)

Parameter

Type: string

Command name.

Return Value

Type: boolean

True if the command is active, false if not.

Example:

  1. var editor1 = new RichTextEditor("#div_editor1");      
  2. editor1.clearHistory();      
  3. console.log("after clear history, can undo : " + editor1.isCommandActive("undo"));
