editor.change event

This event is raised if the content of RichTextEditor is changed by either a programmatic modification or user interaction.

Syntax

function(state,cmd,value)

Example:

  1. editor1.attachEvent("change", function () {      
  2.     var textCount = editor1.getPlainText().trim().length;      
  3.     document.title = "textCount:" + textCount;      
  4. });
