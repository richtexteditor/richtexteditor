selectionchange event

This event is raised if the actual selection in the editor area changes.

Syntax

function(state,cmd,value)

Example:

  1. editor1.attachEvent("change", function () {      
  2.     var textCount = editor1.getPlainText().trim().length;      
  3.     document.title = "textCount:" + textCount;      
  4. });
